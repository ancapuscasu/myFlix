myFlix API Objective

To build the server-side component of a “movies” web application. The web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

Return a list of ALL movies to the user /movies GET None A JSON array holding data about all movies

Allow new users to register /users POST A JSON object holding data about the new user
Example:
{ "FirstName": "Olive", "LastName": "Cat", "Username": "meowmeow", "Email": "olive.cat@gmail.com", "Password": "catsrule", "Birthdate": "2016-03-18" } 		A JSON object holding data about the new user, with an id
Example:
{ "_id" : ObjectId("xxxxxxxxxxxxxxxx"), "FirstName" : "Olive", "LastName" : "Cat", "Username" : "meowmeow", "Email" : "olive.cat@gmail.com, "Password" : "catsrule", "Birthdate" : ISODate("2016-03-18T00:00:00Z") }
Allow users to update their user info /users/:Username PUT A JSON object holding data the user wishes to update with at least one of the following fields: FirstName, LastName, Username, Email, Password, Birthdate.
Example:
{ "FirstName" : "Olivia", "Password" : "secretpassword" } 		A JSON object holding the user's updated data.
Example:
{ "FirstName" : "Olivia", "LastName" : "Cat", "Username" : "meowmeow", "Email" : "olive.cat@gmail.com, "Password" : "secretpassword", "Birthdate" : ISODate("2016-03-18T00:00:00Z") }
Allow users to add a movie to their list of favorites /users/:Username/movies/:MovieID POST None A JSON object holding data about a user with an updated FavouriteMovies list
Example:
{ "_id": "618586c5ec86a4a75b2169a6", "FirstName": "Valentino", "LastName": "Morose", "Username": "valmor", "Email": "valentino.morose@gmail.com", "Password": "Secret Password", "Birthdate": "2004-04-05T00:00:00.000Z", "FavouriteMovies": [ "6182de309c3e0b19943474dc" ], "__v": 0 }
Allow users to remove a movie from their list of favorites /users/:Username/movies/:MovieID DELETE None A JSON object holding data about a user with an updated FavouriteMovies list
Example:
{ "_id": "618586c5ec86a4a75b2169a6", "FirstName": "Valentino", "LastName": "Morose", "Username": "valmor", "Email": "valentino.morose@gmail.com", "Password": "Secret Password", "Birthdate": "2004-04-05T00:00:00.000Z", "FavouriteMovies": [], "__v": 0 }

Example: "Your account was successfully deleted."
