myFlix API Objective

To build the server-side component of a “movies” web application. The web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

BUSINESS LOGIC URL HTTP METHDOD REQUEST BODY DATA FORMAT RESPONSE BODY DATA FORMAT
Return a list of ALL movies to the user /movies GET None A JSON array holding data about all movies
Return data (description, genre, director, image URL, whether it’s featured or not) about a single movie by title to the user /movies/:title GET None A JSON object holding data about a single movie, containing the title, genre, director and release year.
Example: { title: 'Parasite', genre: ['Drama', 'Mystery & Thriller', 'Comedy'], director: 'Bong Joon-ho', releaseYear: 2019 }
Return data about a genre (description) by name/title (e.g., “Thriller”) /movies/:title/genre GET None A text message indicating the genre(s) of a single movie
Example: "Parasite is a Drama, Mystery & Thriller and Comedy."
Return data about a director (bio, birth year, death year) by name /movies/directors/:name GET None A text message indicating data about a director
Example: "Bong Joon-ho was born on September 14, 1969 in Daegu, South Korea. He is a writer and producer, known for Snowpiercer (2013), Okja (2017) and Parasite (2019).""
Allow new users to register /newUser POST A JSON object holding data about the new user
Example: { firstName: 'Olive', lastName: 'Cat' username: 'meowmeow', password: 'catsrule', emailAddress: 'olive.cat@gmail.com' } 		A JSON object holding data about the new user, with an id
Example: { id: 158 firstName: 'Olive', lastName: 'Cat' username: 'meowmeow', password: 'catsrule', emailAddress: 'olive.cat@gmail.com' }
Allow users to update their user info (username) /newUser/:id/info PUT A JSON object holding data about the updated user information
Example: { firstName: 'Olive', lastName: 'Cat' username: 'olivecat', password: 'meowmeow', emailAddress: 'olive.cat@gmail.com' } 		A text message indicating the user information was updated
Example: "Your username and password were successfuly updated."
Allow users to add a movie to their list of favorites /newUser/:id/favourites POST A JSON object holding data about a single movie, containing the title, genre, director and release year.
Example: { title: 'Parasite', genre: ['Drama', 'Mystery & Thriller', 'Comedy'], director: 'Bong Joon-ho', releaseYear: 2019 } 		A text message indicating the user successfully added a new movie to their favourites
Example: "Shrek II was successfully added to your favourites."
Allow users to remove a movie from their list of favorites /newUser/:id/favourites DELETE None A text message indicating the user successfully removed a movie from their favourites
Example: "Shrek II was successfully removed from your favourites."
Allow existing users to deregister /newUser DELETE None A text message indicating the user successfully removed a movie from their favourites
Example: "Your account was successfully deleted."
